# Independent Consultant Management System

## Overview

The Independent Consultant Management System (IMCCS) is a relational database project designed to manage independent healthcare consultants, hospital departments, consultant availability, and consultation requests.

## Features

- Hospital Management
- Consultant Management
- Department Management
- Consultant Availability
- Consultation Request Management
- Payment Tracking
- SQL Retrieval Queries
- Database Normalization

## Technologies Used

- PostgreSQL
- SQL
- Database Management System (DBMS)
